Instrucciones:

Bienvenido a LeidersRP. Por favor, sigue estas instrucciones detalladamente:

  1. Pulsa el botón "Crear Nuevo Usuario".
  2. Introduce la ID de Discord cuando se te solicite.
  3. Rellena cuidadosamente todos los campos requeridos:
    • ➦ ID de PlayStation (Ejemplo: ByRuby12_YT)
    • ➦ Nombre (Ejemplo: Toronto)
    • ➦ Apellido (Ejemplo: Cano)
    • ➦ Sexo (Ejemplo: Masculino o Femenino)
    • ➥ Nacionalidad (Ejemplo: Española)
    • ➥ Fecha de Nacimiento (Ejemplo: 00/00/0000)
    • ➥ Trabajo (Ejemplo: Policia)
    • ➥ Rango (Ejemplo: Superintendente)
  4. Haz clic en el botón "Enviar" para agregar el usuario y los datos.

Una vez completados todos los pasos, recibirás una notificación confirmando que el usuario y los datos se han agregado correctamente. Si encuentras alguna dificultad durante este proceso, te instamos a contactar a la administración para recibir asistencia. Nuestro equipo estará encantado de ayudarte.
